Hazel for Windows: The 5 Best Alternatives

July 7, 2026·8 min read

Quick answer

Hazel is Mac only and no Windows version exists. The closest direct equivalent is File Juggler, which watches folders and applies rules just like Hazel. The free option is DropIt. And if the reason you wanted Hazel was clean, organized folders without the ongoing effort, Filect gets you there without writing rules at all: its AI reads your files and organizes them by what they contain.

Hazel has been the gold standard of Mac file automation for over 15 years. You point it at a folder, define rules ("if the name contains invoice, move it to Finance and rename it by date"), and it quietly keeps your Mac organized forever.

Windows users have been asking for a version just as long, and Noodlesoft has been clear: Hazel is Mac only, and that isn't changing. So if you've switched to a PC, or you read about Hazel and wanted that set-and-forget cleanup on Windows, here are the five tools that actually do the job, including one approach Hazel itself can't match.

1. File Juggler: The Closest Direct Equivalent

Price: Around $40, one-time (free trial available)
Platform: Windows only
What it does: Watches folders you choose and executes condition-based rules: move, rename, delete, or sort files the moment they appear.

If you want Hazel's exact workflow on Windows, this is it. File Juggler's rule engine works the same way Hazel's does: conditions plus actions, running silently in the background. It can even read the text inside PDFs to match rules, which mirrors Hazel's content matching, one of its most loved features.

Best for: Former Hazel users who liked writing rules and want the identical mental model on Windows.

Limitation: Like Hazel, it only handles what your rules anticipate. Every new file pattern means another rule to write, and a messy backlog of old files still needs rules crafted for it.

2. DropIt: The Free Option

Price: Free, open source
Platform: Windows only
What it does: A floating target on your desktop. Drop files on it and your rules sort them. It can also monitor folders on a schedule, Hazel-style.

Best for: Anyone who wants basic rule-based sorting at zero cost and doesn't mind a dated interface.

Limitation: Development is largely dormant, the rule options are simpler than Hazel's, and there is no content matching. It sorts by name, extension, size, and date.

3. Filect: Organization Without the Rules

Price: $15/month (10-day free trial)
Platform: Windows and Mac
What it does: Uses AI to read and understand every file, organizes them automatically by what they contain, and makes everything findable with natural language search.

Here's the honest question for any Hazel switcher: did you love Hazel because you enjoy writing rules, or because you wanted organized files? If it's the second one, AI has changed what's possible since Hazel was designed. Filect doesn't need rules because it reads the actual content of each file. An invoice gets recognized as an invoice whether or not "invoice" is in the filename, something rule-based tools fundamentally can't do.

It also solves the problem rules never touched: finding things. Type "the contract we signed with the landlord in March" and the right file appears, even if it's called document_final_v2.pdf. OCR makes screenshots and scans searchable too.

Best for: People who wanted Hazel's outcome (clean, organized, findable files) without the setup and upkeep of a rule library.

Limitation: It's a subscription rather than a one-time license, and it doesn't do Hazel-style custom actions like scripted renaming chains.

4. AI File Sorter: Free Local AI

Price: Free, open source
Platform: Windows, Mac, and Linux
What it does: Runs AI models on your own hardware to analyze files and suggest categories, which you approve before anything moves.

Best for: Privacy-focused users with capable hardware who want AI categorization with zero cloud involvement.

Limitation: Technical setup, slow without a good GPU, no background folder watching, and no search. See our Filect vs AI File Sorter comparison for details.

5. PowerShell + Task Scheduler: The DIY Route

Price: Free, built into Windows
What it does: A short PowerShell script can move files by extension, age, or name pattern, and Task Scheduler runs it every night.

Best for: Technical users who want full control and no third-party software. A 20-line script covers the classic "sort my Downloads folder by file type" job. Our guide to automatically sorting files into folders covers this approach.

Limitation: You're the developer now. Edge cases, maintenance, and silent failures are all yours, and it can't understand content any more than the rule tools can.

Rules or AI: Which Approach Fits You?

Hazel's rule model rewards a specific kind of person: someone whose files arrive in predictable patterns and who enjoys building a system. If your bank statement always downloads with the same name format, a rule handles it perfectly, forever, for a one-time price. File Juggler gives you exactly that on Windows.

The rule model breaks down when files are unpredictable: client documents named by other people, scans from your phone, screenshots, one-off PDFs. No rule anticipates them, so they pile up unsorted, which is usually why the desktop got messy in the first place. That's the case where AI-based organization earns its subscription: it handles the files you couldn't have written rules for.

The Bottom Line

There's no Hazel for Windows, but there are two good paths forward. If you genuinely like the rule-writing workflow, File Juggler is a faithful equivalent and DropIt is the free version of the same idea. Its AI reads your files, sorts them by real content, and lets you search your whole drive by describing what you remember. For most people landing on this page, that's the better trade.
